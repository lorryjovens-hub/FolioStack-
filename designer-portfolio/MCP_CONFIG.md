# MCP 服务器配置

## 配置说明

根据要求，需要在以下位置之一添加 MCP 服务器配置：

1. **项目级配置**：`.trae/mcp.json`
2. **全局配置**：`~/.cursor/mcp.json` (Trae 兼容配置目录)

## 配置内容

```json
{
  "mcpServers": [
    {
      "name": "pixso",
      "url": "http://127.0.0.1:3667/mcp",
      "type": "sse"
    }
  ]
}
```

## 手动配置步骤

### 方法 1: 项目级配置
1. 打开文件资源管理器
2. 导航到项目根目录
3. 确保 `.trae` 目录存在（如果不存在请创建）
4. 在 `.trae` 目录中创建 `mcp.json` 文件
5. 复制上面的配置内容到文件中

### 方法 2: 全局配置
1. 打开文件资源管理器
2. 导航到用户主目录（`C:\Users\你的用户名`）
3. 确保 `.cursor` 目录存在（如果不存在请创建）
4. 在 `.cursor` 目录中创建 `mcp.json` 文件
5. 复制上面的配置内容到文件中

## 验证配置

配置完成后，Trae 应该能够自动检测到并连接到指定的 MCP 服务器。
