# 艺术作品集PDF风格生成技能库 (skills.md)

## 文件说明

本文件为AI生成艺术作品集PDF提供完整的风格定义和技能参数，可直接用于提示词工程、AI设计工具调用和自动化生成流程。

---

## 通用生成规范

- 输出格式：PDF/A标准，支持打印和电子阅读

- 分辨率：300DPI，CMYK色彩模式（印刷）/ RGB色彩模式（电子）

- 页面尺寸：A4(210×297mm)，边距15-20mm

- 字体：每种风格最多使用3种字体，建立清晰的标题/正文/注释层级

- 交互：支持目录跳转、页码导航、外部链接

### 1. 极简主义风 (Minimalist)

**核心视觉特点**：

- 大量留白(占比40%-60%)

- 单一主色调+黑白灰辅助色

- 无多余装饰元素

- 严格的网格系统排版

- 信息层级极度清晰

- 强调内容本身而非形式

**AI生成技能**：

```Plain Text

- 网格系统设计(12列/16列)
- 负空间精准运用
- 字体层级规划(H1-H6)

- 图片统一裁剪与对齐
- 软件模拟：InDesign、Figma

```
